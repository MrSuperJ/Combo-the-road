## 练级之路 

我的demo repo目录搞得太复杂了。  
新建一个repo，此repo存放页面仿制。  
初级版本 仅 gulp + BrowserSync。

#### 使用方法 

1. 更改gulpfile 变量 workingtarget，为正在制作的项目
2. 执行 `gulp serve`, 启动BrowserSync，使用多屏幕liveReload开发。
